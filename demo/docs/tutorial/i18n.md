If you're using i18n feature, please add `tsdoc.json` file within the installed directory.

`tsdoc.json` should contain `tagDefinitions` of i18n tags you're using.

```json tsdoc.json
{
  "tagDefinitions": [
    {
      "tagName": "@ko",
      "syntaxKind": "inline",
      "allowMultiple": true
    }
  ]
}
```
